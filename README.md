# 钓鱼邮件识别教学游戏

## 项目简介
这是一款模拟真实邮箱环境的互动教学游戏，帮助用户学习如何识别各类钓鱼邮件，提升网络安全意识。

## 快速开始
1. **首次使用**：运行 `init_database.bat` 初始化数据库（当您使用管理员登录游戏后可以在网页创建游戏关卡，这是准备的初始关卡）
2. **启动游戏**：运行 `start.bat` 启动完整应用
